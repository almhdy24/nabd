Nabd – Blood Donation Sudan

Nabd is a mobile application built using React Native and Firebase to connect blood donors with patients and hospitals in real time across Sudan.

The idea of the app is to make it easier and faster to find compatible blood donors during emergencies, instead of depending on manual searching or social media posts.

The system supports two types of users, donors and requestors, with a simple Arabic-first interface and full support for real-time updates. It includes blood type matching, location and hospital filtering, direct contact options, and notification system to keep users updated instantly.

The project was built with focus on real world use, especially in urgent medical situations, and designed to reduce delay in reaching donors when every minute matters.

Technologies used include React Native (Expo), Firebase, and related modern mobile development tools.

Nabd means pulse, representing life and the idea that every response can make a difference.
